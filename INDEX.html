Про Luaro

Luaro - це інноваційний додаток для iOS, який допомагає користувачам контролювати час, проведений на пристрої, та формувати здорові звички використання технологій.

Додаток використовує Family Controls framework для забезпечення ефективного блокування додатків та моніторингу активності, допомагаючи користувачам досягати балансу між цифровим та реальним життям.

Ключові можливості

🔐

Розумне блокування

Створюйте персоналізовані сесії блокування з різними кроками розблокування: дихальні вправи, фізичні вправи, питання для роздумів та календар життя

📈

Детальна аналітика

Відстежуйте час використання додатків, переглядайте статистику та аналізуйте свої звички для кращого розуміння цифрового життя

🧘

Усвідомлені паузи

Дихальні вправи та вправи для роздумів допомагають зробити паузу перед використанням додатків, що сприяє більш свідомому використанню пристрою

⏱️

Гнучкі налаштування

Налаштовуйте тривалість блокування, вибирайте конкретні додатки та категорії, створюйте розклад для автоматичного запуску сесій

🎨

Інтуїтивний інтерфейс

Сучасний та зручний дизайн, який робить управління часом простим та приємним

🔄

Каталог кроків

Збережені шаблони кроків розблокування для швидкого створення нових сесій та повторного використання ефективних практик

Технічна інформація

Використання Family Controls

Додаток використовує Family Controls framework для:

  • Моніторингу активності додатків через DeviceActivityMonitor
  • Блокування доступу до обраних додатків та категорій
  • Створення персоналізованих екранів блокування
  • Збору статистики використання для аналітики

Системні вимоги

  • iOS 16.0 або новіша версія
  • Сумісність з iPhone та iPad
  • Підтримка темної теми
