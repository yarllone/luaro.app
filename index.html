Screen Time Awareness

About the App

Luaro helps users better understand how they spend time on their devices. The app visualizes screen time data and provides insights that encourage more mindful device usage.

How Luaro Uses Screen Time APIs

Luaro uses Apple’s Screen Time APIs to display device usage information and create visual insights about daily habits.

All data is processed locally on the device.

Privacy

Luaro does not collect, store, or transmit device usage data to external servers. All Screen Time information remains on the user’s device.
